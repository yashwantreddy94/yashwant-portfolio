## Hosting on GitHub Pages

To host this portfolio on GitHub Pages:

1. Create a new repository on GitHub.
2. Upload the project files to the repository.
3. Ensure the main homepage file is named `index.html`.
4. Go to **Settings > Pages** in the repository.
5. Select the branch and root folder for deployment.
6. Save the settings and wait for GitHub Pages to publish the site.
